Router Descriptions

Home

HTTP Methods Paths Descriptions
GET / or /router This router takes you to the Home Page of the API

Users

HTTP Methods Paths Descriptions
GET /users This router shows all the users in the database
GET /user/:id This router shows a single user when a user requests a ID
POST /register This router allows us to register a new user into the database
PATCH /user/:id This router allows us to update a users details in the database
POST /user This router allows us to login to the website
DELETE /user/:id This router allows us delete user and their details in the database

Items

HTTP Methods Paths Descriptions
GET /items This router shows all the items in the database
GET /item/:id This router shows a single item when a user requests the item by ID
POST /items This router allows us to add a new item into the database
PACTH /item/:id This router is used to update a item and its details on the database
DELETE /item/:id This router is used to delete a item from the database

Cart

HTTP Methods Paths Descriptions
GET /user/:id/carts This router shows all the items in the cart
POST /user/:id/cart This router allows us to add a new items into the cart
PACTH /user/:id/cart/:id This router is used to update an item from the cart
DELETE /user/:id/cart This router is used to delete all items from the cart
DELETE /user/:id/cart/:id This router is used to delete a single item from the cart
